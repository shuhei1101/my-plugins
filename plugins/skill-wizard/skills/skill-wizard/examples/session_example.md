# skill-wizard セッション管理 — 具体例

スキル作成・編集セッションは `~/.claude/skill-memory/skill-wizard/history/` 配下のファイルで記録する。

## セッションファイルの例

**ファイル名**: `20260426143052_skill_creation.md`

```markdown
# スキル作成セッション - 20260426 143052

## 開始時刻
2026-04-26 14:30:52

## 現在のステータス
ステップ2.5: ステップ一覧の提案

## 対象スキル
- 名前: wiki-wizard
- タイプ: 対話形式
- 出力先: c:/Users/shuhe/repo/my-plugins/plugins/wiki-wizard/skills/wiki-wizard/SKILL.md

## スキル概要
プロジェクトのドキュメントを Wiki 化し、Issue 駆動の意思決定運用と決定経緯の履歴記録を支援するスキル。

## 質問事項（確認済み）
- Q1: Wiki の初期化フロー → フォルダスキャン + 3点セット生成
- Q2: Issue 番号の採番 → Issues.md と イシュー履歴.md 両方から最大番号を取得して +1

## ステップ構成案
1. メインメニュー（フェーズ0）
2. Wiki 構造の初期化（フェーズ1）
3. 新規 Issue 追加（フェーズ2）
4. Issue 決定・転記（フェーズ3）
5. 重複チェック（フェーズ4）
6. wiki.md 最新化（フェーズ5）

## 詳細設計済みステップ
- フェーズ0: 完了
- フェーズ1: 完了
- フェーズ2: 完了
- フェーズ3: 設計中

## 完了情報
- 完了時刻: -
- 出力パス: -
```
