## おまけ機能: ランダムスキル案生成（ステップ1.2 詳細）

この機能はオプション（おまけ）機能。通常フローに必須ではないため、詳細はこのファイルで管理する。

### ステップ1.2(ランダムスキル案生成)
条件: メインメニューで「ランダムでスキル案を生成する」が選択された場合
入力: -
AI実施タスク:
  - **`/idea` を使用**して、複数の観点からスキル案を生成
    - トピック: 「今あったら便利なスキル」
    - 観点: implementation、docs、project、review などから5〜8個を自動選択
    - 各ペルソナが1つずつスキル案を提案（スキル名、対象ユーザー、解決する課題、主な機能）
  - **`/idea` が利用できない場合**（ideaスキルが存在しない、または読み込めない場合）:
    - サブエージェントを直接並列起動（5〜8体）
    - ペルソナファイルも無い場合: 「コスト重視」「品質重視」「工数最小」「ユーザー体験重視」など、自分で観点を設定
    - タスク: 「あなたは{観点名}の視点で、今あったら便利なスキルを1つ提案してください」
  - 各観点からの提案を収集して一覧化する
出力:
  - 「スキル案を生成しました！」
  - スキル案の一覧を表示
  - 「気になる案があれば番号を選んでください。そのスキルを一緒に作りましょう！」
選択肢:
  1. 気になるスキル案を選ぶ（番号で指定） → ステップ1.3（スキルタイプの確認）へ
  2. もう一度ランダム生成する → このステップを再実行
  3. 自分でスキル概要を入力する → ステップ1.3（スキルタイプの確認）へ
  4. その他（入力してください）

補足:
  - 選択されたスキル案は、スキル名と概要が自動的にセットされてステップ1.3（スキルタイプの確認）に渡される
  - ランダム性を出すため、ペルソナ選択と各ペルソナへの指示に「予想外の視点」を含めることを推奨
  - 生成されたスキル案はあくまで「たたき台」。ユーザーが自由にアレンジ・カスタマイズできることを伝える
