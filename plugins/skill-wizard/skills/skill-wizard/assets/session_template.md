# セッション履歴テンプレート

## 基本情報

- スキル実行開始時にセッション履歴ファイルを生成する
- 形式: マークダウン
- ファイル名: `YYYYMMDDHHMMSS.md`
- 保存先: `{対象スキルフォルダ}/sessions/`（固定、変更不可）
- Git管理: sessions/フォルダ内に`.gitignore`を配置し、セッション履歴ファイル（*.md）をGit管理対象外にする
- 記録内容: 日時、スキル名、モード、ステップごとの概要（入力・出力・実行内容・選択した選択肢）
- 目的: 次回セッションで前回の続きから再開できるようにする

## テンプレート

```markdown
# セッション履歴: {スキル名}
日時: {YYYY-MM-DD HH:mm:ss}
モード: 新規作成 / 編集

## ステップ1.1: {ステップタイトル}
- 入力: {ユーザー入力の要約}
- 実行内容: {AIが行ったことの要約}
- 出力: {出力の要約}
- 選択: {ユーザーが選んだ選択肢}

## ステップ1.7: ...
```
