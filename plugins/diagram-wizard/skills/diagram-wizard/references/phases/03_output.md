# フェーズ4: 出力

---

## ステップ4.1（出力先の提案）

**条件**: 図の作成・確認が完了（ステップ3.2またはステップ3.3から遷移）

**入力**: 作成済みの図（全粒度レベル分）、現在のディレクトリ情報

**AI実施タスク**:
  - 現在のディレクトリ構成を確認し、プロジェクトの状況を把握する
  - 以下の候補から最適な出力先を推論し、**理由付き**で提案する:
    - **カレントディレクトリに `.md` ファイルとして出力**: プロジェクトに紐づかない単独の図の場合
    - **プロジェクトの `tmp/` フォルダに出力**: 一時的な図・下書きの場合
    - **既存の `README.md` に追記**: プロジェクト概要に図を追加する場合
    - **`README.md`（概要）と `docs/` フォルダ（詳細）に分割出力**: 複数粒度の図がある場合
    - **ユーザー指定のパス**: 上記のいずれにも該当しない場合
    - **既存ファイルへの追記**: 特定の既存ドキュメントに図を追加する場合
  - ファイル名を提案する（図の内容を反映した命名。例: `architecture-overview.md`, `er-diagram.md`）
  - 複数の図がある場合は、各図の出力先をまとめて提案する

**出力**:
  - 推奨出力先とその理由
  - ファイル名の提案
  - 複数図がある場合は出力先一覧を表形式で提示:

  | No. | 図の名前 | 粒度 | 出力先 | ファイル名 |
  | --- | -------- | ---- | ------ | ---------- |
  | 1   | 全体構成 | 概要 | README.md | （追記） |
  | 2   | 認証モジュール詳細 | 詳細 | docs/ | auth-detail.md |

**選択肢**:
  1. 提案通りに出力する → ステップ4.2へ
  2. 出力先を変更する（パスを指定してください）→ ステップ4.2へ
  3. ファイル名を変更する → ステップ4.2へ
  4. 全部まとめて1ファイルに出力する → ステップ4.2へ
  5. その他（入力してください）

**補足**:
  - `docs/` フォルダが存在しない場合は作成の確認をユーザーに取る
  - 既存ファイルへの追記の場合は、追記位置（末尾 or 特定セクションの後）もあわせて確認する
  - `.md` ファイルにはMermaidコードブロック（` ```mermaid `）で図を埋め込む
  - プレーンテキスト（フォルダ構成図・アスキーアート）の場合はコードブロック（` ``` `）で埋め込む

## ステップ4.2（ファイル出力）

**条件**: ステップ4.1で出力先が確定

**入力**: 作成済みの図、確定した出力先パス、ファイル名

**AI実施タスク**:
  - 確定した出力先に図を書き込む
    - 新規ファイルの場合: ファイルを作成し、図の内容を書き込む
    - 既存ファイルへの追記の場合: 指定位置に図を挿入する（既存内容を破壊しないよう注意）
  - Markdownの見出し構造を適切に設定する（図のタイトル、説明文、コードブロック）
  - 複数ファイルへの出力が必要な場合は順次書き込む
  - 出力完了後、ファイルパスを報告する

**出力**:
  - 出力完了メッセージ
  - 出力先の絶対パス一覧
  - 「ファイルを確認してください」という案内

**選択肢**:
  1. 確認OK → ステップ4.3へ
  2. 出力内容に問題がある → ステップ3.1へ戻る
  3. その他（入力してください）

**補足**:
  - ファイル書き込みに失敗した場合はエラー内容を表示し、別の出力先を提案する
  - 既存ファイルへの追記時は、追記前にファイルの現在の内容を確認し、重複がないことを検証する

## ステップ4.3（次のアクション）

**条件**: ステップ4.2でファイル出力が完了

**入力**: なし

**AI実施タスク**:
  - セッションの継続・終了を確認する
  - 関連する図の作成を提案する（例: クラス図を作成した場合、シーケンス図やER図も必要ではないかと提案）

**出力**:
  - 完了メッセージ
  - 今回のセッションで作成した図の一覧サマリー
  - 関連する図の提案（あれば）

**選択肢**:
  1. 別の図を作成する → ステップ1.1（メインメニュー）へ戻る
  2. 今作った図を編集する → ステップ5.1へ
  3. 終了する
  4. その他（入力してください）

**補足**:
  - 連続セッション時は、前回の図のコンテキスト（プロジェクト情報、用語、スタイル設定）を引き継ぐ
  - 終了時にセッション中に作成した全ファイルのパス一覧を最終表示する
