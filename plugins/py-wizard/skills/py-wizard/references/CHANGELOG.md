## v1.3.5 (2026-04-11)

### 追加

- **仮想環境（venv）のサポート**: プロジェクトごとの独立したPython環境を推奨
  - `setup_venv.bat` - venv作成と依存関係インストールの自動化
  - `activate.bat` - venv有効化のショートカット
  - すべての `.bat` ファイルで venv 自動有効化機能（venv存在時）
  - `.gitignore` に `venv/`, `env/`, `ENV/` を追加
  - 場所: `references/python_rules.md` のフォルダ構成テンプレート

- **外部ツールの自動インストール機能**: 可能な限り自動インストールを試みる
  - インストール済みチェック機能
  - 自動インストール（winget等のパッケージマネージャー使用）
  - 失敗時は手動インストールガイドにフォールバック
  - 例: ffmpegの場合、`winget install -e --id Gyan.FFmpeg` で自動インストール
  - 場所: `references/python_rules.md` の外部ツール自動インストール

### 変更

- **セットアップ方法の改善**: README生成時に venv 使用を推奨するインストール手順に変更
  - レガシーなグローバル環境インストールも選択肢として残す
  - セットアップ確認セクションの追加
