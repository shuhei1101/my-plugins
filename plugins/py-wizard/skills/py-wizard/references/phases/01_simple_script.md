# フェーズ: 簡易スクリプト作成

単一 `.py` ファイルで完結するスクリプト作成フロー。

---

## ステップS.1（要件ヒアリング）

**条件**: 簡易スクリプト作成モードが選択された場合

**入力**: ユーザーが第一次情報を自由入力

**AI実施タスク**:
- ユーザーの入力から以下の情報を洗い出す
- 未回答項目をヒアリングシート形式で質問する

**ヒアリングシート**:

```
QS1. スクリプトのファイル名は？（例: convert_csv.py）
→

QS2. このスクリプトは何をするもの？（1〜2行で）
→

QS3. コマンドライン引数・オプションはありますか？
→ (例: --input input.csv --output output.json --verbose / なし)

QS4. 使用する外部ライブラリ・APIはありますか？（標準ライブラリのみならなし）
→

QS5. 出力の形式は？
→ (ファイル / 標準出力 / なし)

QS6. 作成先ディレクトリは？
→ (カレントディレクトリ / パス指定)
```

**出力**:
- ヒアリングシート（既知の項目を埋めた状態で提示）
- 💡 **Tips**: 入力時は `/copy` コマンドを使用して入力フォームを活用してください

**選択肢**: なし（自由入力で回答してもらう）

**補足**:
- 全項目が埋まったらステップS.2へ
- 未回答・情報不足がある場合はこのステップを再実行して再確認
- 引数なしスクリプトの場合は QS3 は「なし」でOK

---

## ステップS.2（ライブラリ調査・確認）

**条件**: ステップS.1のヒアリングが完了した場合

**AI実施タスク**:
- QS4 でユーザー指定ライブラリがある場合: そのまま採用
- 指定がない or 不足している場合: **MCP（context7）やWeb検索を必ず実行**して最新情報を調査
  - 代替候補を複数リストアップ（各2〜3個）
  - 各技術の特徴・メリット・デメリットを簡潔にまとめる
  - 1つをAI推奨として選定・理由を1〜2行で記載
- 標準ライブラリのみで完結する場合: このステップをスキップしてS.3へ

**出力**:
- ライブラリ選定案（`[ユーザー指定]` / `[AI推奨]` ラベル付き）
- 「この構成で合っていますか？」

**選択肢**:
1. OK → ステップS.3へ
2. ライブラリを変更する（候補から選択 or 別のライブラリを指定）→ 修正して再提示
3. その他（入力してください）

**補足**:
- フルパッケージの技術調査（ステップ2.6a/2.6b）と同じ調査基準で実施する

---

## ステップS.3（スクリプト生成）

**条件**: ステップS.2完了後（またはS.2スキップ時）

**AI実施タスク**:
- `references/python_rules.md` の「簡易スクリプトルール」に従いスクリプトを生成する
- ファイルヘッダー形式を必ず含める（後述）
- 生成後に `~/.claude/skill-memory/py-wizard/{YYYYMMDDHHMMSS}_session.md` に記録

**生成するファイル**:
- `{script_name}.py` — スクリプト本体（1ファイルのみ）

**ヘッダー形式（ファイル冒頭に必須）**:

```python
"""
{script_name} — {一行説明（端的に何をするか）}

Usage:
  # コマンドライン引数の説明
  python {script_name}.py [options] {positional_args}

"""
```

**コード骨格（python_rules.md の「簡易スクリプトルール」参照）**:

```python
"""...(ヘッダーdocstring)..."""

# ── 標準ライブラリ ──────────────────────────────────────────
...

# ── サードパーティ ──────────────────────────────────────────
...

# ── 定数 ────────────────────────────────────────────────────
...

# ── プライベート関数 ─────────────────────────────────────────
def _helper(...) -> ...:
    ...

# ── メイン処理 ──────────────────────────────────────────────
def main(args: argparse.Namespace) -> None:
    ...

def parse_args() -> argparse.Namespace:
    ...

if __name__ == "__main__":
    args = parse_args()
    main(args)
```

**出力**:
- 生成したスクリプトのプレビュー（全文）
- 作成先パス
- 「このスクリプトで合っていますか？」

**選択肢**:
1. OK → 完了（ステップ5.1へ）
2. 修正内容を伝える → 修正して再提示
3. 実装・改修フェーズに移行する → ステップ3.1へ
4. その他（入力してください）

**補足**:
- 引数なしスクリプトの場合: Usage のコメント行を省略し `python {script_name}.py` のみ記載
- 外部ライブラリのインポートが必要な場合、コメントで `pip install {package}` を明示
- 複雑なロジック・深いネスト・難解な式には適宜インラインコメントを追記（docstringとは別に）
