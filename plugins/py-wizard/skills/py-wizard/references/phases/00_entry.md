## ステップ1.1（モード選択）

**条件**: スキル起動時。ただしユーザーが起動時に意図を明示している場合はAIが自動判定してジャンプ:
- 「新規プロジェクト作成したい」→ ステップ2.1へ
- 「〇〇を実装したい」「〇〇を修正したい」→ ステップ3.1へ
- 上記いずれにも該当しない場合 → 通常のモード選択を表示

**入力**: ユーザーのリクエスト全文

**AI実施タスク**:
- 起動時コンテキスト分析（ユーザーの入力から意図を判定）
- `~/.claude/skill-memory/py-wizard/` ディレクトリが存在するか確認し、前回セッションがあれば再開オプションを提示
- プロジェクト専用スキルが存在するか確認（改修モードの場合に読み込むため）

**出力**:
- py-wizardスキルの説明（新規プロジェクト作成・実装改修ができること）
- 前回セッションがある場合はその旨を表示

**選択肢**:
1. 新規Pythonプロジェクトを作成する → ステップ2.1へ
2. 既存プロジェクトを実装・改修する → ステップ3.1へ
3. 前回セッションを再開する（該当がある場合のみ表示）
4. その他（入力してください）

**補足**:
- 自動判定で分岐した場合は「〇〇モードと判断しました」と明示する
- 前回セッションのmemoryファイルは `~/.claude/skill-memory/py-wizard/{YYYYMMDDHHMMSS}_session.md` に保存
