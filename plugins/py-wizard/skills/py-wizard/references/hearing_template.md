# 要件ヒアリングテンプレート

ステップ2.1で使用する固定質問テンプレート。既知の項目は埋めた状態で提示し、AIおすすめがある項目は `→ (おすすめ: XXX)` で提示する。

---

```
Q1. プロジェクト名は？（パッケージ名にも使用）
→ 

Q2. このツールは何をするもの？（1〜2行で）
→ 

Q3. 主な機能を箇条書きで教えてください
→ 

Q4. ターゲットユーザーは？
→ (自分用 / チーム / 一般公開)

Q5. 使用する外部ライブラリ・APIはありますか？
→ 

Q6. GUI/CLIどちらを使う？
→ (両方 / GUIのみ / CLIのみ)

Q7. 複数モードはありますか？（例: 変換ツールならmp4→mp3、mp4→aviなど）
→ 

Q8. 設定項目として外部に出したいパラメータはありますか？
→ 

Q9. 入力データの形式は？
→ (ファイル / テキスト / API / なし / 複数)

Q10. 出力データの形式は？
→ (ファイル / テキスト / API / GUI表示 / なし / 複数)

Q11. 連携する他ツール（別パッケージ）はありますか？ある場合はツール名とデフォルト配置先を教えてください
→ 
```

💡 **Tips**: 入力時は `/copy` コマンドを使用して入力フォームを活用してください

---

## 追加質問ガイドライン

固定質問の回答をもとに、以下の観点で追加質問が必要か判断する:

- **モードが複数ある場合**: 各モードの詳細（入出力、使用ライブラリ）
- **連携ツールがある場合**: 連携方法、データ受け渡し形式
- **外部APIがある場合**: 認証方法、レート制限、エラーハンドリング
- **大量データを扱う場合**: input/output/cacheフォルダの要否
