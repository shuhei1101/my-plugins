# イシュー履歴 — 決定済みイシューの履歴ログ

決定したイシューの履歴を追記型で残すドキュメント。決定経緯の追跡 + 「なぜこの決定をしたか」を後から振り返るための資産。

**運用ルール**:
- 新規イシューは [Issues.md](Issues.md) に登録
- ユーザーが決定 → 機能別ドキュメントに転記 → Issues.md から削除 → このファイルの最下部に追記
- イシュー番号は連番（ISSUE-001, ISSUE-002, ...）、一度発行した番号は使い回さない
- 最新の決定が一番下に来る（追記型・時系列順）

**追記フォーマット**:
```markdown
## ISSUE-XXX: {タイトル}

- **質問内容**: {元の質問の概要、1〜2 行}
- **決定した内容**: {採用した選択肢と理由の要約、1〜3 行}
- **決定日**: YYYY-MM-DD
- **転記先**: [{ドキュメント名}](ファイル名.md#{セクション名})

---
```
