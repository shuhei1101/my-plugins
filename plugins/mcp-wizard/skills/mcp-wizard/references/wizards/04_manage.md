# フェーズ5: MCP管理

---

## ステップ10（一覧表示）

**条件**: メインメニューで「登録済みMCPを管理する」が選択された場合

**入力**: -

**AI実施タスク**:
- 現在登録済みのMCPサーバーを取得する:
  - `claude mcp list` をBashで実行
  - `.mcp.json`（プロジェクト）があれば読み込む
  - `~/.claude/settings.local.json`（ユーザー）があれば読み込む
- 取得結果を統合し、スコープ情報付きで一覧表示する
- 登録数が0件の場合、登録フローへの誘導を提案する

**出力**:
- 登録済みMCPの一覧:

| No. | サーバー名 | 接続タイプ | コマンド | スコープ | 状態 |
|-----|-----------|-----------|---------|---------|------|
| 1 | playwright | stdio | npx -y @anthropic/mcp-server-playwright | user | 有効 |
| 2 | github | stdio | npx -y @modelcontextprotocol/server-github | project | 有効 |
| ... | ... | ... | ... | ... | ... |

- 0件の場合: 「登録済みのMCPサーバーはありません」

**選択肢**:
- MCPが1件以上ある場合:
  1. MCPを選択して操作する（番号で指定） → ステップ11へ
  2. 新しいMCPを追加する → ステップ7へ
  3. メインメニューに戻る → ステップ1へ
  4. その他（入力してください）
- 0件の場合:
  1. MCPを登録する → ステップ7へ
  2. おすすめMCPを探す → ステップ5へ
  3. メインメニューに戻る → ステップ1へ
  4. その他（入力してください）

**補足**:
- CLI結果と設定ファイルの両方から取得し、重複はマージする
- 設定ファイルが見つからない場合はCLI結果のみで表示する

---

## ステップ11（操作選択）

**条件**: ステップ10でMCPが番号指定された場合

**入力**: 選択されたMCPの情報（サーバー名・接続タイプ・コマンド・スコープ等）

**AI実施タスク**:
- 選択されたMCPの詳細情報を表示する
- 実行可能な操作の一覧を提示する

**出力**:
- 選択したMCPの詳細:
  ```
  【選択中のMCP】
  サーバー名: playwright
  接続タイプ: stdio
  コマンド: npx -y @anthropic/mcp-server-playwright
  引数: なし
  環境変数: なし
  スコープ: user
  状態: 有効
  ```

**選択肢**:
1. 設定を編集する → 編集対象の項目を質問し、変更後ステップ12へ
2. 削除する → ステップ12へ
3. 有効/無効を切り替える → ステップ12へ
4. 別のMCPを選択する → ステップ10へ
5. メインメニューに戻る → ステップ1へ
6. その他（入力してください）

**補足**:
- 編集の場合、変更したい項目（サーバー名・コマンド・引数・環境変数・スコープ）を質問してから確認ステップへ
- 有効/無効の切り替えは `/mcp enable/disable` コマンドに対応

---

## ステップ12（確認と実行）

**条件**: ステップ11で操作が選択された場合

**入力**: 選択された操作（編集 / 削除 / 有効無効切替）、対象MCP情報

**AI実施タスク**:
- 操作内容を表形式で確認を求める
- 承認後に実行する:
  - **編集**: 設定ファイルをReadで読み込み、Editで該当箇所を更新
  - **削除**: `claude mcp remove {サーバー名}` を実行、または設定ファイルから該当エントリを削除
  - **有効/無効切替**: `/mcp enable {サーバー名}` or `/mcp disable {サーバー名}` に対応する処理

**出力**:
- 操作内容の確認:
  - 編集の場合:

    | 項目 | 変更前 | 変更後 |
    |------|--------|--------|
    | コマンド | npx | node |
    | 引数 | -y @anthropic/... | /path/to/local/... |

  - 削除の場合:
    ```
    以下のMCPを削除します:
    サーバー名: playwright
    スコープ: user
    ```
  - 有効/無効切替の場合:
    ```
    以下のMCPを無効化します:
    サーバー名: playwright
    現在の状態: 有効 → 無効
    ```

**選択肢**:
1. 実行する → 実行後ステップ13へ
2. 修正する → ステップ11へ戻る
3. キャンセルする → ステップ10へ
4. その他（入力してください）

**補足**:
- 削除操作は不可逆なため、「本当に削除しますか？」の確認を必ず挟む
- 編集時はCLIでは直接編集できないため、設定ファイルのEdit操作で行う
- 実行後、`claude mcp list` で変更が反映されたことを確認して結果を表示する
