## ステップ1（メインメニュー）

**条件**: スキル起動時。ただし、ユーザーのリクエストからモードが明確な場合は該当ステップに自動ジャンプ:
- 「MCPのおすすめ教えて」「おすすめMCP」→ ステップ5へ
- 「MCP登録したい」「MCPを追加して」→ ステップ7へ
- 「MCP管理」「MCP一覧」「MCP削除したい」→ ステップ10へ
- 「最新のMCP情報」「MCPトレンド」「MCP情報更新」→ ステップ2へ
- 「MCPって何」「MCPとは」→ ステップ1.5へ

**入力**: ユーザーのリクエスト全文

**AI実施タスク**:
- リクエスト内容を分析し、上記の自動ジャンプ条件をチェック
- いずれにも該当しない場合、メインメニューを提示
- MCP初心者向けに、MCPとは何か（1〜2文）と各モードの簡単な説明を添える

**出力**:
- MCPの簡単な説明（「MCPはClaude Codeに外部ツール・サービスを接続するための仕組みです」等）
- 各モードの説明付きメニュー:
  - 最新MCP情報を取得する: 「公式レジストリ等から最新のMCPサーバー情報を取得し、カタログに保存します」
  - おすすめMCPを探す: 「あなたのスキルやプロジェクトに合ったMCPサーバーを探します」
  - MCPを登録する: 「MCPサーバーをClaude Codeに登録します（CLIまたは設定ファイル）」
  - 登録済みMCPを管理する: 「登録済みのMCPサーバーを一覧・編集・削除します」
  - MCPって何？: 「MCPの仕組みやAPIとの違いをわかりやすく説明します」

**選択肢**:
1. 最新MCP情報を取得する → ステップ2へ
2. おすすめMCPを探す → ステップ5へ
3. MCPを登録する → ステップ7へ
4. 登録済みMCPを管理する → ステップ10へ
5. MCPって何？ → ステップ1.5へ
6. その他（入力してください）

**補足**:
- 自動ジャンプ時はメニュー表示をスキップし、該当モードのステップへ直接遷移
- `mcp-catalog/` にデータがない状態でステップ5（おすすめ探索）に進もうとした場合、「まず最新情報を取得しましょう」とステップ2への誘導を提案する

## ステップ1.5（MCPとは？）

**条件**: メインメニューで「MCPって何？」が選択された場合

**入力**: -

**AI実施タスク**:
- `references/mcp_intro.md` を読み込み、MCP初心者向けに説明を表示する

**出力**:
- `references/mcp_intro.md` の内容を表示する

**選択肢**:
1. わかった、メインメニューに戻る → ステップ1へ
2. もっと詳しく知りたい（質問してください）
3. その他（入力してください）

**補足**:
- 選択肢2の場合、ユーザーの質問に回答してから再度この選択肢を提示する
- 説明は初心者が「なるほど」と思える粒度に留める。技術的な深掘りは質問があった場合のみ
