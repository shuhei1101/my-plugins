# フェーズ4: MCP登録

---

## ステップ7（登録方法選択）

**条件**: メインメニューで「MCPを登録する」が選択された場合、またはステップ6から登録フローに遷移した場合

**入力**: おすすめ探索から来た場合はMCP情報（MCP名・URL・概要）

**AI実施タスク**:
- おすすめ探索からの遷移かどうかを判定する
  - 遷移の場合: 選択されたMCPの情報を表示し、登録方法の選択へ
  - 直接来た場合: 登録したいMCPの情報入力を促す
- 2つの登録方法を初心者向けに説明する

**出力**:
- おすすめ探索からの場合: 「以下のMCPを登録します」と確認表示
- 登録方法の説明:
  ```
  【方法1: CLIコマンド（おすすめ）】
  `claude mcp add` コマンドで登録します。
  最も簡単で、コマンド1つで完了します。

  【方法2: 設定ファイル直接編集】
  `.mcp.json` や `settings.json` を直接編集します。
  複数のMCPをまとめて設定したい場合や、細かい設定をしたい場合に便利です。
  ```

**選択肢**:
1. CLIコマンドで登録する（おすすめ） → ステップ8へ
2. 設定ファイルを直接編集する → ステップ8へ
3. 戻る → 遷移元に応じてステップ6 or ステップ1へ
4. その他（入力してください）

**補足**:
- 初心者には方法1（CLI）を推奨
- 選択した方法はステップ8で使用する

---

## ステップ8（情報入力）

**条件**: ステップ7で登録方法が選択された場合

**入力**: 登録方法（CLI or 設定ファイル）、おすすめ探索からの場合はMCP情報

**AI実施タスク**:
- おすすめ探索から来た場合:
  - MCP名・コマンド・引数等を自動入力し、確認を求める
  - URLからインストール方法を推定する（npm, pip, docker等）
- 直接来た場合:
  - 対話的に必要情報を収集する
- 以下の情報を順番に確認する

**出力**:
- 必要情報の入力フォーム（多角度質問形式）:
  ```
  Q1. サーバー名は？
  → （例: playwright, github, context7）

  Q2. 接続タイプは？
  → (stdio / HTTP / SSE)

  Q3. コマンドは？
  → （例: npx, node, python）

  Q4. 引数は？
  → （例: -y @anthropic/mcp-server-playwright）

  Q5. 環境変数は？（不要なら「なし」）
  → （例: GITHUB_TOKEN=xxx）

  Q6. スコープは？
  → (user: 全プロジェクト共通 / project: このプロジェクトのみ)

  💡 **Tips**: 入力時は `/copy` コマンドを使用して入力フォームを活用してください
  ```
- おすすめ探索からの場合は自動入力済みの内容を表示:
  ```
  以下の内容で自動入力しました。修正があれば番号で指定してください。
  Q1. サーバー名: playwright
  Q2. 接続タイプ: stdio
  Q3. コマンド: npx
  Q4. 引数: -y @anthropic/mcp-server-playwright
  Q5. 環境変数: なし
  Q6. スコープ: user
  ```

**選択肢**: なし（質問への回答に集中してもらう）

**補足**:
- おすすめ探索からの自動入力の場合、ユーザーは「OK」だけで次に進める
- 回答漏れがある場合は未回答項目のみ再確認する
- 環境変数にAPIキー等が必要な場合、その取得方法も案内する
- 全質問に回答がそろったらステップ9へ進む

## ステップ9（確認と実行）

**条件**: ステップ8で全情報が揃った場合

**入力**: ステップ8で収集した登録情報、登録方法（CLI or 設定ファイル）

**AI実施タスク**:
- 入力内容を表形式で整理して確認を求める
- 登録方法に応じて実行コマンド or 編集内容のプレビューを生成する
- 承認後に実行する:
  - **CLI**: `claude mcp add {サーバー名} --scope {スコープ} -- {コマンド} {引数}` を実行
  - **設定ファイル**: 対象ファイル（`.mcp.json` or `settings.json`）を読み込み、`mcpServers` に追記

**出力**:
- 登録内容の確認表:

  | 項目 | 値 |
  |------|-----|
  | サーバー名 | playwright |
  | 接続タイプ | stdio |
  | コマンド | npx |
  | 引数 | -y @anthropic/mcp-server-playwright |
  | 環境変数 | なし |
  | スコープ | user |
  | 登録方法 | CLI |

- 実行コマンド/編集内容のプレビュー:
  - CLI: `claude mcp add playwright --scope user -- npx -y @anthropic/mcp-server-playwright`
  - 設定ファイル: 変更前後のdiffを表示

**選択肢**:
1. この内容で登録する → 実行後ステップ13へ
2. 修正する（項目を指定） → ステップ8へ戻る
3. 続けて別のMCPも登録する → 実行後ステップ8へ（登録方法は前回と同じ設定を引き継ぐ）
4. キャンセルする → ステップ1へ
5. その他（入力してください）

**補足**:
- CLI実行時はBashツールで実行し、成功/失敗を確認して結果を表示する
- 設定ファイル編集時は既存のMCP設定を壊さないよう、Readで現在の内容を確認してからEditで追記する
- 環境変数にAPIキー等の機密情報が含まれる場合、設定ファイルに直書きせず環境変数参照を推奨する
- 選択肢3で連続登録が可能
