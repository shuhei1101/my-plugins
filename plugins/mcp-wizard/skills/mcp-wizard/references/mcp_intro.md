# MCPとは？

MCPは「Model Context Protocol」の略で、Claude Codeに外部ツール・サービスを接続するための仕組みです。

## イメージ

USB規格のようなもの。中身はデータ通信ですが、規格が統一されているからどのデバイスも「挿すだけ」で使えます。MCPも同じで、登録するだけでAIが自動的にツールを発見して使えるようになります。

## APIとの違い

| | API | MCP |
|---|---|---|
| **通信** | HTTPリクエスト/レスポンス | stdin/stdout（ローカル）or HTTP/SSE（リモート） |
| **誰が呼ぶ** | 開発者がコードで呼ぶ | Claude（AI）が自動で判断して呼ぶ |
| **認証** | APIキー等を毎回渡す | 起動時に環境変数で渡せば後は自動 |
| **目的** | アプリ間の通信 | AIに「できること」を増やす |

一言でいうと、MCPは「AIが使える道具箱を増やす仕組み」です。

## 具体例

例えば Playwright MCP を登録すると、Claude Code が以下のツールを自動で使えるように:
- `browser_navigate` — URLに移動
- `browser_click` — 要素をクリック
- `browser_screenshot` — スクリーンショット撮影

個別の設定は不要。登録するだけで全ツールが使えます。

## 接続タイプ

- **stdio**: ローカルでプロセスを起動して通信（最も一般的）
- **HTTP**: リモートサーバーにHTTPで接続
- **SSE**: リモートサーバーにServer-Sent Eventsで接続
