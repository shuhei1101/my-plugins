# フェーズA: エントリーポイント

---

## ステップ1(モード選択)

**条件**: スキル起動時。ただし、ユーザーのリクエストから意図が明確な場合は該当ステップに直接ジャンプする（スキップ条件は補足に記載）

**入力**: ユーザーのリクエスト全文

**AI実施タスク**:
- **起動時のコンテキスト分析**（このステップを実行するか判定）:
  - リクエスト内容から、新規作成/編集/一覧/会話/エージェント間会話/カテゴリ管理のいずれかの意図を検出する
  - 意図が明確な場合は該当ステップに直接ジャンプ（このステップをスキップ）
  - 意図が不明確な場合はメニューを表示する
- `~/.claude/skill-memory/agent-wizard/{YYYYMMDDHHMMSS}_agent_session.md` を作成しセッション開始を記録
- `references/agent_file_spec.md` を参照し、エージェントファイルの仕様を把握

**出力**:
```
🤖 Agent Wizard — 何をしますか？

1. エージェントを新規作成する
2. 既存エージェントを編集する
3. エージェント一覧を見る
4. エージェントと会話する（1対1）
5. エージェント同士で会話させる
6. カテゴリフォルダを管理する
7. その他（入力してください）
```

**選択肢**: 上記メニューの通り

**補足**:
- **ジャンプ条件**:
  - 「エージェント作りたい」「ペルソナ作成」等 → ステップ2へ
  - 「{エージェント名}を修正/編集/変更」「{パス}を編集」等 → ステップ9へ
  - 「エージェント一覧」「agents見せて」等 → ステップ11へ
  - 「{エージェント名}と話したい」等 → ステップ13へ
  - 「エージェント同士で議論させて」等 → ステップ14へ
  - 「フォルダ整理」「カテゴリ分けして」等 → ステップ12へ
- **選択肢による遷移**:
  - 選択肢1 → ステップ2（スコープ・配置先確認）
  - 選択肢2 → ステップ9（編集対象の選択）
  - 選択肢3 → ステップ11（エージェント一覧表示）
  - 選択肢4 → ステップ13（1対1会話）
  - 選択肢5 → ステップ14（エージェント間会話）
  - 選択肢6 → ステップ12（カテゴリフォルダ管理）
  - 選択肢7 → ユーザーの入力内容をジャンプ条件で再判定し、該当するステップへ遷移する。判定できない場合は再度メニューを表示する
