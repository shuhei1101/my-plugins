# Agent Wizard — エージェントウィザード

## このスキルについて

カスタムサブエージェント（Claude Code の `agents/` に配置する `.md` ファイル）を対話形式で作成・編集・管理するスキル。

- **新規作成**: ペルソナ設計→フロントマター設定→関連リンク収集→回答形式設計→ファイル出力
- **編集**: 全スコープのエージェントを走査し、対話で修正
- **一覧・管理**: スコープ別×カテゴリ別のテーブル表示、フォルダ構成管理
- **1対1会話**: エージェントのペルソナで会話。タスク実行はサブエージェント委譲
- **エージェント間会話**: 複数エージェントにトピックを議論させ、ログをファイル出力

## 対応スコープ

| スコープ | パス | 説明 |
|---|---|---|
| ユーザーレベル | `~/.claude/agents/` | 全プロジェクトで使える |
| プロジェクトレベル | `.claude/agents/` | プロジェクト専用 |
| プラグイン内 | `{plugin}/agents/` | プラグインとして配布可能 |

## Memory活用ルール

セッション情報は `~/.claude/skill-memory/agent-wizard/` に保存する。
プラグインはインストール/アンインストールされるため、スキル配下ではなくホームディレクトリ配下に保存する。

### 保存先
`~/.claude/skill-memory/agent-wizard/` — ディレクトリが存在しない場合は自動作成すること。

### セッション管理
- **エージェント作成/編集**: `{YYYYMMDDHHMMSS}_agent_session.md`
- **会話ログ**: `{YYYYMMDDHHMMSS}_chat_{agent-name}.md`
- **エージェント間会話ログ**: `{YYYYMMDDHHMMSS}_debate_{topic-slug}.md`
- **セッションサマリー**: `{YYYYMMDDHHMMSS}_session_summary.md`

**圧縮コマンドやセッション中断に備え、大事な内容だけを記録すること。**
